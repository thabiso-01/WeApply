# Local Setup Instructions

This document outlines the steps to set up the development environment for the WeApply project locally.

## Prerequisites

- Ensure you have [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) installed.
- Make sure you have [Git](https://git-scm.com/) installed.

## Setup Steps

1. **Clone the repository**:
   
   ```bash
   git clone https://github.com/thabiso-01/WeApply.git
   cd WeApply
   ```

2. **Install dependencies**:
   
   ```bash
   npm install
   ```

3. **Set up environment variables**:
   
   Create a `.env` file in the root directory and add the necessary environment variables. You may refer to `.env.example` for guidance.

4. **Run the application**:
   
   ```bash
   npm start
   ```

5. **Access the application**:
   
   Open your browser and navigate to `http://localhost:3000` to access the application.

## Troubleshooting

- If you encounter issues, verify your Node.js and npm installation. 
- Ensure that all dependencies are correctly installed.

For any further questions, please contact the project maintainer.
