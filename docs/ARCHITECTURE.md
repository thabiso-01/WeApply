# System Architecture

This document outlines the system architecture of WeApply.

## Overview

The WeApply system is designed to facilitate a streamlined application process for users, leveraging various technologies to ensure performance and scalability.

## Components

1. **Frontend**  
   - Built using React for a responsive user interface.
   - Utilizes Redux for state management.

2. **Backend**  
   - RESTful API developed in Node.js.
   - Connects to a MongoDB database for data storage.

3. **Deployment**  
   - Hosted on AWS, using services such as EC2 and S3.

4. **Security**  
   - Implements JWT (JSON Web Tokens) for authentication.
   - Utilizes HTTPS for secure communication.
